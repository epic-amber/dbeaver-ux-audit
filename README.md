# UX Audit · DBeaver Ultimate · Security Preferences

Интерактивный UX-аудит экранов **Preferences → Security** в DBeaver Ultimate 26.1.0 (macOS).

**[Открыть полное исследование в Figma →](https://www.figma.com/design/fye36sRLiAdEAWCDwP4hIx/Test-assignment.-Security-page?node-id=0-1&t=GcWwthzBG4t5Pfpc-1)**

## Что внутри

- **30 UX-проблем** на 6 экранах с интерактивными точками — кликайте, чтобы увидеть описание, severity и ссылки на эвристики
- **Before / After** — сравнение текущего UI и предложенного редизайна с 10 ключевыми изменениями
- **Новый User Flow** — одношаговый диалог задания пароля вместо 3-слойного wizard
- **Таблица решённых проблем** — 16 issues, сгруппированных по severity

## Стек

Один HTML-файл (`index.html`) + папка со скриншотами. Без фреймворков, без сборки.

## Запуск локально

Открыть `index.html` в браузере или запустить любой статический сервер:

```bash
npx serve .
```

## Деплой на Vercel

Проект готов к деплою как статический сайт — Vercel автоматически подхватит `index.html` как точку входа. Никакой конфигурации не нужно.
